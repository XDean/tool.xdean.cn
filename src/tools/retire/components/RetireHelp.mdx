# 功能

作为打工人，你是否想过搬砖的日子什么时候是个头？
本工具帮助你了解：

- 最早何时可以退休
- 现在立刻退休至少需要多少钱
- 你这辈子
  - 一共赚了多少钱
  - 一共花了多少钱
  - 一共留下多少钱

# 基础参数

- 当前存款：存款理财等，不包括房产，房租收入可另算
- 年化利率：存款每年有多少收益，参考值2%
- 退休年龄：男性65，女性60，也可以调小看看提前退休的结果

# 收支项目

- 类型
  - 工作：退休就是为了不工作
  - 收入：工作以外的被动收入
    - 养老金从退休才开始领取，大致估算1万元基数交10年退休金约1200元每月
  - 支出：财富自由的主要障碍
    - 社保：最低缴纳15年，实际退休后就不交了
- 参数
  - 每月金额
  - 涨幅：每年增长，可以设置按比例和按金额（月）
  - 年龄：有些项目有起止年龄，可以不填
  - 范围：项目的金额范围，例如工资不是无限增长的

# 计算结果

- 最早退休年龄：这一年退休，一辈子也花不完存款
- 立即退休需要：就在今天，如果你有这么多钱，按照上面的支出水平，永远花不完
- 按时退休遗产：勤勤恳恳干到正式退休，到死会留下多少
